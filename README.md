
Project Name: Digital Tools Buying Website

1 React.js
2 Tailwind CSS
3 JavaScript (ES6+)
4 JSON (for data handling)
5 React Toastify (for notifications)
Features:
1 Add to Cart Functionality.
2 Dynamic Product Display.
3 Clean & Responsive UI.
